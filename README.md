Universidade Católica de Brasília

📝 Descrição

O projeto SafeCrypto foi desenvolvido seguindo a metodologia Lean Inception e tem como objetivo simular funcionalidades de uma plataforma de gestão de criptomoedas.

Ele permite:

Cadastro de usuários
Operações simuladas de compra e venda de criptoativos
Visualização de relatórios detalhados
Front-end moderno com React e Vite

O projeto prioriza protótipos funcionais e funcionalidades essenciais, alinhadas às boas práticas de Lean Inception.

Requisito mínimo: Node.js ≥ 20

⭐ Principais características
Front-end moderno: Desenvolvido com React e Vite
Gerenciamento de dependências: Node.js e npm
Banco de dados estruturado: Suporta operações de clientes, pacotes de serviços e relatórios detalhados
Lean Inception: Priorização das funcionalidades essenciais, facilitando prototipagem rápida
Auditoria e segurança: Dependências atualizadas, vulnerabilidades corrigidas
Design profissional: Layout inspirado em fintechs, responsivo e intuitivo
